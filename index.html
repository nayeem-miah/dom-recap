DOM এর ধরন (Types of DOM)

DOM মূলত ৩ ধরনের হয়ঃ

  • 1. Legacy DOM: পুরনো ব্রাউজার ভিত্তিক DOM, যা খুব basic ফিচার দেয়।
  • 2. XML DOM: XML ডকুমেন্টের জন্য DOM, যা XML ডাটা ম্যানেজ করে।
  • 3. HTML DOM: HTML ডকুমেন্টের জন্য DOM, যা ওয়েব পেজের HTML উপাদানগুলোকে প্রোগ্রাম থেকে নিয়ন্ত্রণ করে।

DOM থেকে এলিমেন্ট সিলেক্ট করার মেথড

  • getElementById(id)
    একটি নির্দিষ্ট id অনুসারে একটি এলিমেন্ট নির্বাচন করে। এটি একটি single এলিমেন্ট রিটার্ন করে।
    উদাহরণ: document.getElementById('header')
  • getElementsByClassName(className)
    নির্দিষ্ট class নামের সব এলিমেন্টের কালেকশন রিটার্ন করে। এটি একটি HTMLCollection (array-like object) দেয়।
    উদাহরণ: document.getElementsByClassName('menu-item')
  • getElementsByTagName(tagName)
    নির্দিষ্ট ট্যাগ নামের সব এলিমেন্টের কালেকশন রিটার্ন করে। যেমন সব p বা div এলিমেন্ট।
    উদাহরণ: document.getElementsByTagName('p')
